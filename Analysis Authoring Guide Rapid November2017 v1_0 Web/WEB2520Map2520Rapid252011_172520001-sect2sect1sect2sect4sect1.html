Having a group of key stakeholders upon whom to draw is critical to supporting the activity of information gathering. The stakeholder group to be consulted is agreed with the project manager as part of producing the analysis scope document. Stakeholders typically include domain experts, clinicians and users, as well as other analysts. These individuals should be able to provide relevant information or identify key reference sources.
The objective early in the project is to have understanding of the stakeholders and a sufficient set of their needs to document in the analysis model.
