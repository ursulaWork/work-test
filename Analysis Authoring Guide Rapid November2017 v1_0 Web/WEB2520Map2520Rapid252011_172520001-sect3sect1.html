“Agile’ is an umbrella term used to describe a general approach to software development. Though there are many flavours of Agile e.g. eXtreme Programming (XP), Scrum, Kanban, Lean Software Development – all Agile processes promote responsiveness in the forms of adaptive planning, evolutionary development, early delivery, in-line testing and bug-resolution as well as continuous improvement.‘”
