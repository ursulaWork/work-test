“Tagged Values are a UML Extensibility Mechanism, that can be associated with UML Model Entities (e.g. Packages / Elements / Connectors etc.,). They provide additional values for entities used within Models e.g. suggested or example data that can be used during model testing, or for explaining aspects of a model not supported by other mechanisms.”
There are no defined Core Notations for UML Tagged Values as UML Extensibility Mechanisms — they are applied to the various model elements as metadata (Toolset dependent).
