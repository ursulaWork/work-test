The Artefact Set

The ‘Artefact Set’ is the specific collection of Artefacts developed (or to be developed) within the Analysis Model during an Analysis Project. The development of the Artefact Set starts very early in the work, taking its primary goal as meeting the project aims and scope (it may be continually updated throughout the project lifecyle). Each of the approved methodologies defines the artefacts available to Analysis Projects undertaken within its remit; these are mostly diagram-based products, but may also include ‘related documents’ as well as other non-diagrammatic supporting content (e.g. to support an understanding of the domain or management processes). The development order and dependencies of various products / artefacts are outlined in the section: Artefact Product Map.

A defined set of Artefacts are developed during an Analysis Project undertaken in accordance with the Rapid Methodology; these are tied to specific project stage, and the final set may vary dependent upon the needs of the project and dependent upon the scope.
