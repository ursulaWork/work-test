“A ‘Theme’ within Agile is an organising principle or categorisation of User Stories. Themes are often used to organise stories into releases so that various sub-teams can work on them.”
An example of a Theme could be: within a university registration system there might be themes around students, course management, transcript generation, grade administration, financial processing.
