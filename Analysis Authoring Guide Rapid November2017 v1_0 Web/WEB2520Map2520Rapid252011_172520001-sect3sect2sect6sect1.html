“Attributes are metadata values that qualify a characteristic or property of an object or element. Attributes are usually attached to an object / element to facilitate the availability of related information, or to preset information values / parameters. The application of Attributes to specific object or element is subject to the modelling approach and methodology being employed.”
Attributes are typically applied to Elements as metadata; they can also be represented on Diagrams as part of the Element they define. Example figures are available (where appropriate) as part of the definitions for specific Elements.
