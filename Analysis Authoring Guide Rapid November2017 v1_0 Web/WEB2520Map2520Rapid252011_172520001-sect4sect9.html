Heading Details
Product Name
Stakeholder Analysis - Personas
Description
“Stakeholder Analysis - Personas are fictional characters created as part of a Stakeholder Analysis process to represent different user types or views. They are a way of representing a stakeholder or set of stakeholders.”
Additional Description
Stakeholder Analysis is the analysis of the levels of power and interest of stakeholders in order to assess the weight that should be attached to their issues.
Purpose
The purpose of Stakeholder Analysis is to provides a means of categorising stakeholders in order to identify the most appropriate stakeholder management approach; within that ‘Personas’ create a visual representation/ human face for otherwise abstract data about stakeholders and are cognitively compelling to the audience.
